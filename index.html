¿Qué es R?

  • R es un dialecto de un lenguaje llamada S.
  • R es gratuito y de código abierto.
  • Disponible para Windows, MACOSX, Linux, Unix e incluso playstation.
  • De igual forma podemos accesar a R por medio de la nube mediante RStudio cloud.

Características de R

  • R está diseñado especialmente para análisis estadístico y gráficos.
  • Contiene más de 17000 paquetes que incrementan su funcionalidad.
  • Permite realizar además; análisis de series de tiempo, manipulación de datos, aprendizaje maquina, etc..
  • Para descargar R se tiene que dirigir a https://www.r-project.org/

R para macOSX

RStudio

  • RStudio es una interfaz gráfica para R.
  • RStudio tiene una versión gratuita que es potente y suficiente para el curso.
  • RStudio facilita muchos procesos como la visualización de tablas, control de versiones con git, creación de proyectos, etc.
  • Para descargar RStudio se tiene que dirigir a: https://rstudio.com/

Variables en R

  • R como se ha visto funciona como una potente calculadora científica.
  • Sin embargo, la mayor potencia de R está en el análisis estadístico.
  • Para lo anterior se hace uso extensivo de variables.
  • Una variable guarda valores que posteriormente pueden servir para realizar otro cálculo.

Variables en R (cont.)

Estructuras de datos

  • R posee múltiples tipos de estructura de datos, sin embargo, la mayoría surge de los vectores.
  • Los vectores más conocidos son los atómicos, las listas y los data.frames.
  • Comprender la forma en que se crean estos tipos de objetos es importante ya que permite manejarlos con más facilidad.
  • De igual forma su comprensión permite entender los tipos de datos más complejos.
